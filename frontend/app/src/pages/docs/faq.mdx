[open a discussion](https://github.com/pingcap/tidb.ai/discussions).

<FAQBox title="Is this RAG app free?">
- The code of this RAG app is open source, it's free to use;
- The database of this RAG app is free for **25GiB storage** as we use [TiDB Serverless](https://tidb.cloud);
- **But** the token of OpenAI is paid, you should have an account of https://platform.openai.com for an API token.
</FAQBox>

<FAQBox title="Can I use a different chat model?">
  Currently we use OpenAI as chat model by default, and can't change to another one.
  But we plan to add more LLMs support, such as: 

- [Claude3](https://claude.ai/)
- [Gemini](https://gemini.google.com/)
- [Ollama](https://ollama.ai/)
- ...

</FAQBox>

</FAQBox>
